Command: artiax mask triangle radius

Usage:
artiax mask triangle radius [float]

The artiax mask triangle radius command enables the mask triangles mouse mode. If a float is specified, the mouse mode radius is set, meaning only connected triangles within the radius of the pressed one are masked. If no radius is specified, all connected triangles get masked.

Examples:

artiax mask triangle radius
artiax mask triangle radius 10
