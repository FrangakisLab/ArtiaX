Command: artiax geo2vol

Usage:
artiax geo2vol [model-spec] [geomodels model-specs] [subdivideLength value]

The artiax geo2vol command adds the specified geomodels (comma seperated) to the chosen volume by voxelizing the geomodel, as long as it fits inside the bounding box of the volume. If no volume is specified, a new one is created. If no geomodels are specified, all shown ones are used.

property meaning expected value type
geomodels The geomodels to voxelize and add to the volume. A comma seperated list. Can only add geomodels within the bounding box of the volume. If no geomodels are specified, all shown ones are used. comma seperated model-specs
subdivideLength Specifies how finely the geomodels will be voxelized. A smaller value gives a higher resolution, but also takes longer. Specifically, the command subdivides all triangles longer than subdivideLength before voxelizing. Defaults to the smallest pixelsize of the specified volume, or 1 if no volume is specified. float

Examples:

artiax geo2vol #1.1.1 geomodels #1.3.1,#1.3.2
artiax geo2vol subdivideLength 0.5
artiax geo2vol #1.1.2 subdivideLength 0.25
