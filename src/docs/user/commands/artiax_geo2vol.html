Command: artiax geo2vol

Usage:
artiax geo2vol model-spec [geomodels model-specs] [subdivideLength value] [merge Bool]

The artiax geo2vol command turns the specified geomodels (comma seperated) into a volume by voxelizing the geomodel. The geomodel can also be merged with an already existing volume, as long as it fits inside the bounding box of the volume. The new volume is automatically scaled to fit the indicated existing tomogram. If no tomogram is indicated, the currently selected tomogram is chosen. If no geomodels are specified, all shown ones are used.

property meaning expected value type
geomodels The geomodels to voxelize and add to the volume. A comma seperated list. Can only add geomodels within the bounding box of the volume. If no geomodels are specified, all shown ones are used. comma seperated model-specs
subdivideLength Specifies how finely the geomodels will be voxelized. A smaller value gives a higher resolution, but also takes longer. Specifically, the command subdivides all triangles longer than subdivideLength before voxelizing. Defaults to the smallest pixelsize of the specified volume, or 1 if no volume is specified. float
merge Specifies if the volume from the geomodel will be merged with the indicated tomogram or if a new volume will be created. Can only be True if a tomogram is specified. bool

Examples:

artiax geo2vol #1.1.1 geomodels #1.3.1,#1.3.2
artiax geo2vol subdivideLength 0.5
artiax geo2vol #1.1.2 subdivideLength 0.25
artiax geo2vol #1.1.3 geomodels 3.1.1 subdivideLength 0.25 merge True
artiax geo2vol #1.1.3 geomodels 3.1.1 subdivideLength 0.25 merge False
