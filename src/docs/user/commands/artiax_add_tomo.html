Command: artiax add tomo

Usage:
artiax add tomo model-spec

The artiax add tomo command adds a tomogram that has already been opened in ChimeraX to the window and displays it. The options menu of the selected tomogram will not be automatically selected. This command can be used instead of selecting and clicking the required tomogram in the tomogram table.
If another tomogram is opened, it must be hidden beforehand with the command artiax hide.

Examples:

artiax add tomo #2
