Command: artiax moveCameraAlongLine

Usage:
artiax moveCameraAlongLine model-spec [numFrames value] [backwards value] [distanceBehind value] [topRotation value] [facingRotation value] [cameraRotation value] [monoCamera value] [maxAngle value]

The artiax moveCameraAlongLine command moves the camera along a specified line model. Can be used to record movies where the camera moves along a desired path. The number of frames, as well as the facing of the camera can be specified, along with an optional smoothing option. It is recommended to use to the curved line options UI to find an appropriate rotation.

property meaning default expected value type
numFrames Sets over how many frames the camera will move along the line. Cannot be larger than the resolution of the line. The number of frames can end up being more than the specified value if smoothing is turned on with the 'maxAngle' keyword. resolution of the line int
backwards If the camera moves forwards or backwards along the line. false bool
distanceBehind The camera is put the specified distance behind the line. 500 float
topRotation Rotates the top of the camera. 0 float
facingRotation Rotates the facing of the camera. 0 float
cameraRotation Rotates the camera. 0 float
monoCamera If true, changes the camera mode to 'mono'. true bool
maxAngle The maximum allowed angle for each frame of the camera. Linearly interpolates between frames if the angle is too large. If None, a suitable angle is estimated to avoid 'bumps'. Set to 0 to avoid smoothing. None None or float

Examples:

artiax moveCameraAlongLine #1.3.1 numFrames 300 distanceBehind 0 maxAngle 0
artiax moveCameraAlongLine #1.3.2 numFrames 500 topRotation 30 facingRotation 42 cameraRotation 15 maxAngle 0.1
