Command: artiax particles

Usage:
artiax particles [ model-spec ] [radius value] [axesSize value] [surfaceLevel value] [color colorname] [originScaleFactor value] [transScaleFactor value]

The artiax particles command enables setting a property of the selected particle list. A blank spec will change the property on all particle lists currently open.

property meaning expected value type default value
radius radius of the particle sphere (Angstrom) float 4
axesSize size of the xyz axes (Angstrom) float 15
color color of particle colorname N/A
surfaceLevel Isosurface threshold for density map associated with this particle list float ?
originScaleFactor Scaling factor for particle origins contained in the particle list. float 1
transScaleFactor Scaling factor for particle alignment shifts contained in the particle list. float 1

Examples:

artiax particles radius 8
artiax particles #1.2.1 color blue
artiax particles #1.2.2 origin 5
