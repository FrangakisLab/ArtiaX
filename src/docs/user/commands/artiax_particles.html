Command: artiax particles

Usage:
artiax particles [ model-spec ] property value

The artiax particles command enables setting a property of the selected particle list. A blank spec will change the property on all particle lists currently open.

property meaning expected value type default value
radius radius of the particle sphere float 4
axes size of the xyz axes float 15
color color of particle sphere string N/A
origin sets the origin scale factor float 1
surface dont know float ?
trans sets the shift pixelsize scale factor float 1

Examples:

artiax particles radius 8
artiax particles #1.2.1 color blue
artiax particles #1.2.2 origin 5
