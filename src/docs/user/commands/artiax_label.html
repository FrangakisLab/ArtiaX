Command: artiax label

Usage:
artiax label model-spec attribute [height value] [offset value]

The artiax label command allows for setting a label on each particle in the selected particle list displaying the particles value for the selected attribute. height sets the size of the label, and offset receives a three comma seperated values that determine the offset of the label from the particle center. If no height is given, it is set to 8, and if no offset is given, it is set to 4,4,4.
