Command: artiax colormap

Usage:
artiax colormap model-spec attribute [minValue value] [maxValue value] [transparency value]

The artiax colormap command allows for setting a colormap to the chosen particle lsit. if no minimum or maximum value is chosen the lowest and largest values available in the selected particle list is chosen as minimum and maximum values respectively.
