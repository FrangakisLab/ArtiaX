Command: artiax colormap

Usage:
artiax colormap model-spec attribute [palette palette-name] [minValue value] [maxValue value] [transparency value]

The artiax colormap command allows setting the colormap of the specified particle list based on metadata attributes. If no minimum or maximum value is chosen the lowest and largest values available in the selected particle list is chosen as minimum and maximum values respectively.

Available attribute names for colormapping can be retrieved using the artiax info command.

Examples:

artiax colormap #1.2.1 position_x
artiax colormap #1.2.1 cross_correlation min 0.3 max 1.0
