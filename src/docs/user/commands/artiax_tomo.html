Command: artiax tomo

Usage:
artiax tomo model-spec [contrasCenter value] [contrastWidth value] [contrastMode dol | lod] [slice value] [sliceDirection value] [endSlice value] [slicePerFrame value] [pixelSize value]

The artiax tomo command enables setting a property of the selected tomogram.

property meaning expected value type default value
contrastCenter image value that will be displayed with 50% of greyscale intensity float N/A
contrastWidth range centered around contrastCenter that defines minimum and maximum greyscale intensity. float N/A
contrastMode Whether to render with dark-on-light (default) or light-on-dark contrast str ('dol' or 'lod') dol
slice Slice to display. Slice thickness is the grid step of the volume. integer N/A
sliceDirection normal of the slice plane three comma seperated numbers 0,0,1
endSlice If provided, slices are animated from slice to endSlice integer N/A
slicePerFrame If endSlice is provided, move this many slices per frame of the animation integer N/A
pixelSize Pixel Size in Angstrom float 1

Examples:

artiax tomo #1.1.1 slice 52 
artiax tomo #1.1.4 sliceDirection 1,1,1
artiax tomo #1.1.4 sliceDirection 1,1,1 endSlice 100 slicePerFrame 5
artiax tomo #1.1.4 pixelSize 0.5
