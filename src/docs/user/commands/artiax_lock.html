Command: artiax lock, artiax unlock

Usage:
artiax lock [ model-spec ] [ type ]

Usage:
artiax unlock [ model-spec ] [ type ]

The commands artiax lock and artiax unlock allow restricting the manipulation of the specified particle lists using mouse modes. A blank spec will lock or unlock all particle lists currently open. Locking rotation or translation is particularly useful in VR to prevent moving particles away from the original locations.

The keyword type specifies which type of movement is locked/unlocked. When not specified, type defaults to 'movement'.

type meaning
translation locks particles to their current origin
rotation locks particles in their current orientation
movement completely prevents moving particles

Examples:

artiax lock #1.2.1
artiax unlock #1.2.2 rotation
