Command: artiax open tomo

Usage:
artiax open tomo filename 

The artiax open tomo command opens a tomogram. It works only in combination with the path of the tomogram to be opened. The options menu of the selected tomogram will not be automatically selected.

Examples:

artiax open tomo /home/name/data/tomos/tomo4_HR.em
artiax open tomo /home/name/data/tomos/tomo4_HR.mrc
