Command:artiax close tomo

Usage:
artix close tomo [ internal ID | row number table ]

The artiax close tomo command closes a tomogram by internal ID (not the model ID #{} but the row number of the tomo in the tomo table).

For example: artiax close tomo 2 closes the second tomogram in the tomogram table.
