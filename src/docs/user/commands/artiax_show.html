Command: artiax show, artiax hide

Usage: artiax show [ model-spec ] [ style ]

Usage: artiax hide [ model-spec ] [ style ]

The artiax show and artiax hide commands set the rendering style of the specified particle lists. An empty spec applies the action to all currently open particle lists. The style keyword specifies which rendering style should be displayed or hidden.

If no style is provided, the action is applied to all styles of the specified lists.

style meaning
marker Hide/Show marker representation of the particles.
surface Hide/Show the surface representation of the particles.
axes Hide/Show the axes representation of the particles.

Examples:

artiax hide #1.2.1 markers
artiax show #1.2.2 surfaces
