Command: artiax vol2geo

Usage:
artiax vol2geo [model-specs] [name value]

The artiax vol2geo command creates a new geomodel from the specified models. Only uses the masked triangles. If no model is specified, the currently selected models are used. Works well in combination with the mask triangles mouse mode.

property meaning expected value type
name Sets the name of the created geomodel string

Examples:

artiax vol2geo name volume_geomodel
artiax vol2geo #1,#2,#4
