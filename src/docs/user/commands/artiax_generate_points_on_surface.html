Command: artiax gen on surface

Usage:
artiax gen on surface model-spec method numPts [radius value]

The artiax gene on surface command generates points on the chosen surface. Two different methods of generating points are available. Similar to the artiax gen in surface command.

property meaning expected value type
method Method for generating points. Available methods are: uniform and poisson. The uniform method generates the specified number of points uniformly sampled on the surface, and the poisson method generates points using a poisson disk sampling scheme (meaning points are generated randomly but never within a certain radius of each other) using the specified number of points and radius. Ten times the specified number of points are generated uniformly on the surface, and points are then removed until no points are within the specified radius of each other, meaning that the number of points in the end will approximately be the specified number of points, depending on the size and shape of the surface. string
numPts The number of points to generate on the surface when using the uniform, and a tenth of the number of originally generated points for the poisson method. int
radius The poisson disk radius used for the poisson method. float

Examples:

artiax gen on surface #1.3.1 uniform 100
artiax gen on surface #1.3.1 poisson 50 radius 10
