Command: artiax delete duplicate

Usage:
artiax delete duplicate [ model-spec ] [radius value]

The artiax delete duplicates command allows deleting particle duplicates or particles that are within a specified radius (in angstrom) to each other. Default radius is 0.

Examples:

artiax delete duplicate #1.2.1 radius 1
artiax delete duplicate #1.2.1
