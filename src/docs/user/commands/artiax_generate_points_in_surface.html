Command: artiax gen in surface

Usage:
artiax generate points in surface model-spec method [numP ts value] [radius value]

The artiax gen in surface command generates points inside the closed volume of the chosen surface. Three different methods of generating points are available. Similar to the artiax gen on surface command.

property meaning expected value type
method Method for generating points. Available methods are: uniform, regular grid, and poisson. The uniform method generates the specified number of points uniformly sampled within the bounding box of the surface, the regular grid method points on a regular 3D grid, and the poisson method generates points using a poisson disk sampling scheme (meaning points are generated randomly but never within a certain radius of each other) using the specified radius. All methods generate points withing the bounding box of the surface, and then removes the points that are outside of the enclosed volume of the surface. string
numPts The number of points to generate within the bounding box of the surface when using the uniform , or regular grid method. int
radius The poisson disk radius used for the poisson method. float

Examples:

artiax gen in surface #1.3.1 uniform numPts 100
artiax gen in surface #1.3.1 "regular grid" numPts 500
artiax gen in surface #1.3.1 poisson radius 100
