Command: artiax clip

Usage:
artiax clip ( thickness | off | toggle ) [ model model-spec ]

The artiax clip command allows turning on clipping planes with thickness Angstrom distance above and below the tilted slab position of a tomogram imported in ArtiaX. The model parameter allows specifying the tomogram that the clipping is applied to. If no model parameter is set, the first displayed tomogram is used.

By default, clipped surface models are not capped to ensure performance on any machine. If your machine has a performant CPU, turn on capping using surface cap true.

Examples:

artiax clip 200 #1.1.1
artiax clip 100
artiax clip off
artiax clip toggle
