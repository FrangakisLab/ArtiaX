Command: artiax open

Usage:
open filename path [format type] volume model-spec | (voldim dimensions voxelsize size) [prefix string]

The open command is based on ChimeraX’s open command. It allows you to load various files (such as particle lists, geomodels, etc.) into ArtiaX. You can specify the file, format, and additional options to customize the import process. For further information regarding the underlying structure, check the chimeraX save command documentation.

property meaning expected value type
filename Specifies the path to the file that will be imported string
format Specifies the file format to import (e.g., motl, relion, peet) string

Examples:

open filename /path/to/particles format relion
open filename /path/to/model format peet

Opening a particle list in RELION 5 format requires additional arguments. To compute the coordinates, the dimensions and pixel size of the corresponding tomogram are needed. This information can either be extracted from a tomogram volume that is already loaded in ArtiaX or specified manually. Additionally, to correctly read the 'rlnTomoName' column in the star file, the desired prefix preceding the tomogram number must be specified.

property meaning expected value type
volume Specifies the volume from which tomogram dimensions and pixelsize will be extracted (e.g., #1.1.1) model-spec
voldim Specifies volume dimensions (e.g., 500,500,1000). Alternativly, specify volume List of 3 floats
voxelsize Specifies volume voxelsize (e.g., 8.73). Alternativly, specify volume float
prefix Specifies prefix before tomogram number in 'rlnTomoName' (e.g., TS_, tomo). string

Examples:

open filename /path/to/output format relion5 voldim 896,696,250 voxelsize 11.52 prefix tomo
open filename /path/to/output format relion5 volume #1.1.1 prefix tomo
