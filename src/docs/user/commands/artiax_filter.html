Command: artiax filter

Usage:
artiax filter model-spec low-pass length high-pass length [lpd value] [hpd value] [unit value] [lpCutoff value] [hpCutoff value] [threshold value]

The artiax filter command lets the user apply low-pass, high-pass, and band-pass filters to a specified tomogram. Gaussian, raised cosine, and box decays are available.

property meaning default expected value type
low-pass length The low-pass cutoff length, in pixels or angstrom depending on the unit. If no low-pass component is required for the filter, set to 0 (in combination with a non-zero low-pass decay). The low-pass length cannot be 0 when the unit is set to angstrom, unless the low-pass decay is also set to 0 or "None", in which case no low-pass component is used. N/A float
high-pass length The high-pass cutoff length, in pixels or angstrom depending on the unit. If no high-pass component is required for the filter, set to 0 (in combination with a non-zero high-pass decay). The high-pass length cannot be 0 when the unit is set to angstrom, unless the high-pass decay is also set to 0 or "None", in which case no high-pass component is used. N/A float
lpd The low-pass decay length in pixels. If set to 0, a box filter is used. If set to "None", the default value is used. Cannot be used when using angstrom as the unit, but is always set automatically in that case. low-pass cutoff / 4 float
hpd The high-pass decay length in pixels. If set to 0, a box filter is used. If set to "None", the default value is used. Cannot be used when using angstrom as the unit, but is always set automatically in that case. high-pass cutoff / 4 float
unit The unit for which the other values are entered. "pixels" and "angstrom" are available. When using angstrom the cutoff lenghts cannot be set to 0, and the decays can not be set manually. pixels string
lpCutoff The cutoff method used for the low-pass component. "gaussian" and (raised-)"cosine" are available. If the low-pass decay is set to 0, a box filter is used instead. If a gaussian cutoff is selected, the values below the threshold value set by the "threshold" parameter are set to 0. gaussian string
hpCutoff The cutoff method used for the high-pass component. "gaussian" and (raised-)"cosine" are available. If the high-pass decay is set to 0, a box filter is used instead. If a gaussian cutoff is selected, the values below the threshold value set by the "threshold" parameter are set to 0. gaussian string
threshold The value used as the threshold for the gaussian cutoff method. 0.001 float

Examples:

artiax filter #1.1.1 200 50
artiax filter #1.1.4 0 40 hpd 20 hpCutoff cosine
artiax filter #1.1.2 50 0 unit angstrom
artiax filter #1.1.2 150 40 lpd 0 threshold 0.000001
