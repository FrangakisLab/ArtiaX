Command: artiax triangulate

Usage:
artiax triangulate [furthestSite value]

The artiax triangulate command creates links between the selected particles using Delaunay triangulation. Useful in combination with the triangles from links command. The furthestSite option decides whether only the convex hull of the particles should be created, True, or whether internal triangles should also be created, False. The default value is True.

Examples:

artiax triangulate
artiax triangulate furthestSite False
