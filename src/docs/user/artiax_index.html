• Tools
    •
    • ArtiaX  – Tool for Tomogram and Particle list model management.
    • ArtiaX Options  – Tool for Tomogram and Particle list manipulation.
    • ArtiaX Toolbar  – ArtiaX toolbar tab.
  • Commands
    •
    • add tomo  – add a already opened tomogram
    • close tomo  – close a tomogram
    • hide  – hide surfaces, axes or markers i.e.
    • open tomo  – open a tomogram
    • particles attach  – attach particles to a motivelist
    • show  – show surfaces, axes or markers i.e.
    • start  – start artiax
    • view  – view different camera positions
