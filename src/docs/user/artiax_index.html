ArtiaX User Guide

ArtiaX is an open-source extension for visualization and processing of cryo electron tomography data. It allows rendering of macromolecular complexes within their context in electron tomograms, picking particles for subtomogram averaging, and assessing results of image classification or template matching procedures.

For help on the user interface refer to the Tools section. All commands need to be prefixed with "artiax".

  •  Commands
    • add tomo  – add a already opened tomogram
    • attach  – attach a surface to a particle lsit
    • boundary – create a boundary around the currently selected particles
    • colormap  – set a colormap for a particle list
    • fit line – fit a line through currently selected particles
    • fit sphere – fit a sphere to currently selected particles
    • fit surface – fit a surface through currently selected particles
    • flip – flip the selected particles around the given axis
    • geomodel color – set the color of a geomodel
    • info  – get information about a particle list or tomogram
    • label  – add a label with the value of an attribute to every particle in a list
    • lock, unlock  – lock or unlock the rotation, translation or movement of particles
    • mask – create a mask from a geometric model
    • open tomo  – open a tomogram
    • particles  – set a property of particles
    • remove links – remove all links connected to currently selected particles
    • select inside – select all particles inside currently selected model
    • show, hide  – show or hide surfaces, axes or markers i.e.
    • start  – start artiax
    • tomo  – set a property of a tomogram
    • triangles from links – creates triangles between all currently selected particles linked into triangles
    • triangulate – links currently selected particles into triangles
    • view  – view different camera positions
