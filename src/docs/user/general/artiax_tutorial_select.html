General: Displaying and selecting subsets using commands

Using the Selection/Display box in the Select/Manipulate tab of the ArtiaX Options tool window, one can select or show/hide particles based on user defined metadata thresholds. This is also possible using the command line interface of ChimeraX, since all metadata are stored as attributes of the MarkerSet associated with a particle list.

Example

  1. Download the demo data, unpack and change to the empiar_10304 directory using the cd command in ChimeraX.
  2. Open the provided tomogram of the EMPIAR-10304 dataset (tomogram/empiar_10304_tomo_10_SART.em) in ArtiaX

    artiax open tomo tomogram/empiar_10304_tomo_10_SART.em

  3. Set the tomogram pixelsize to 8.4 Angstrom, either in the user interface (ArtiaX Options->Physical Coordinates->Pixel Size) or using the command:

    artiax tomo #1.1.1 pixel 8.4

  4. Open the provided particle list (particle_list/empiar_10304_tomo_10_particles.em) in ArtiaX

    open particle_list/empiar_10304_tomo_10_particles.em format motl


  5. Set the particle list pixelsize to 8.4 Angstrom, either in the user interface (ArtiaX Options->Select/Manipulate->Pixelsize Factors Origin) or using the command:

    artiax particles #1.2.1 originScaleFactor 8.4

  6. To select all particles with cross-correlation values larger than 0.5 use the following command:

    select #1.2.1@@cross_correlation>0.5

  7. To select particles based on multiple metadata values (cross_correlation > 0.5, part_number < 500), chain attribute selection like this:

    select #1.2.1@@cross_correlation>0.5 & #1.2.1@@part_number<500

  8. Use the same commands to selectively show and hide particles:

    hide all
    show #1.2.1@@cross_correlation>0.5 & #1.2.1@@part_number<500
