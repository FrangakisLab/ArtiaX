Tool: ArtiaX

The Artiax Window is used to handle tomograms and particle lists. It lets the user see information about the models, show or hide them, decide which model to change the options for using the Artiax Options tab, and create, save, load and add models.

  • Tomograms:

    The tomograms box contains a table with all the loaded tomograms which shows their IDs, names, if they're being displayed or not, and if they are the currently selected "options" tomogram or not. The box also contains buttons for opening a tomogram, closing the currently highlighted tomogram, as well as adding tomograms already loaded into ChimeraX to ArtiaX, allowing them to be manipulated using the Tomogram tab in the Artiax Options tab.

  • Particle lists:

    The particle list box contains a table with all the loaded particle lists which shows their IDs, names, if they're being displayed or not, and if they are the currently selected "options" particle list or not. The box also contains buttons for creating a new, empty particle list, opening a saved particle list, as well as saving or closing the currently highlighted particle list.
