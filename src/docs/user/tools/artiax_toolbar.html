Tool: Tutorial (Qt)

The Tutorial (Qt) tool demonstrates how to create a graphical tool interface using the Qt for Python (PySide2) toolkit.

The tool displays a type-in field. When the user hits the Return key, any text in the field will be shown in the log as its HTML equivalent. That means that the text: 

	<a href="http://www.cgl.ucsf.edu/chimerax">ChimeraX home page</a>
will be logged as:

ChimeraX home page

Choosing Clear frmo the tool context menu will clear the text field
