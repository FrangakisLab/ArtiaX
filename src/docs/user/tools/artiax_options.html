Tool: Options

The ArtiaX Options tab contains the tools and setting for visualizing and changing tomograms and particle lists. The tab is itself divided into three tabs, Tomogram, Visualization, and Select/Manipulate. The tomogram or particle list that is affected by the changes made in the Options window is decided by which tab is being used and which particle list or tomogram is selected as "Options" in the Artiax window.

  • Tomogram:

    The Tomogram tab is used to manipulate the currently selected "options" tomogram.

    • Physical Coordinates:

      Set the pixel size of the tomogram or volume if it isn't specified in the file header.

    • Contrast settings:

      Used to set the center and width of a linear contrast slope, used as a quick, reproducible way of changing the contrast settings. Contrast can still be adjusted using the Volume Viewer box if desired. By pressing the "Invert Contrast" button, a new tomogram is created for which the density values are inverted.

    • Navigation:

      Changes the slice direction and which slice is currently shown. The "Slice Direction" lets the user input a vector used as a normal vector to the slice plane. The "Slice" slider can then be used to select which slice to show. The "<" (Shortcut: F3) and ">" (Shortcut: F4) jumps backward/forwards one slice at a time, while "<<" (Shortcut: F7) and ">>" (Shortcut: F8) jump 10 slices at a time.
      TIP: Use the mouse-wheel while hovering the mouse over the slider to change the slice currently shown.

    • Orthoplanes:

      Sets the camera to view the tomogram from one of the standard directions. Also changes the slice direction so that the slice plane is parallel to the viewing plane.

    • Processing:

      Includes different options for tomogram processing. In the "Average" section, the user can create a copy of the tomogram averaged in the set direction, using the number of nearest slabs for the averaging. In the "Filtering" area, the user can create a copy of the tomogram that is filtered using a Low Pass or High Pass filter. A Gaussian or Cosine decay can be selected. If the unit is set to pixels and the pass frequency is set to zero the center of decay will be at zero. If decay size is set to zero a box filter is used. If the unit is set to "angstrom", the decay size is always set to 1/pass*0.25.

    • Tomogram arithmetics:

      GUI incorporation of arithmetic operations for tomograms. The tomogram for which the ArtiaX options tab is currently activated is the first operand. A second operand can be selected as well as the desired operation (Addition, Subtraction, Multiplication, Division). After pressing "Compute", a new tomogram will be generated from the result of this operation.

  • Visualization:

    The Visualization tab is used to change the display options for the currently selected "options" particle list, and any associated surface model.

    • Color Settings:

      Used to change the color and transparency of the particles in the particle list. If "Single Color" is selected all particle will get the selected color. If "Colormap" is used, particles can be colored according to their value with respect to some attribute.

    • Marker/Axes Display:

      Changes the size of the markers and axes on the particles.

    • Surface Display:

      Add a density map to be rendered at each particle location. One can either choose a map already opened in ChimeraX, or choose a file to be loaded. Using the Surface Level slider, the isosurface threshold of the map can be manipulated.

  • Select/Manipulate:

    The Select/Manipulate tab allows for selecting or showing particles based on attributes, and scaling and manipulation particles.

    • Scaling:

      Allows setting the units of the location and/or alignment shift values in the particle list. For example, if the particle list contains positions in pixels of a reference tomogram, all positions can be scaled to physical units (Angstrom) using the pixel size of the tomogram. The same applies to alignment shifts.

    • Manipulation:

      Used to delete or reset particle to their original positions. The "Remove Duplicates" section allows for deleting particle duplicates or particles that are within a specified radius (in angstrom) to each other.

    • Selection/Display:

      Lets the user select or display particles based on some attribute. Multiple selectors can be used simultaneously. Particularly useful when a particle list contains information from more than one tomogram or when the particles are numbered. Also works well with the "Manipulation" options.

  • Reorient:

    The Reorient tab allows to reorient the particles.

    • Reorient Particles:

      Reorients the particle according to their order in the particle list, so that every particle points to the next one in the particle list.

    • Create new, reordered particle list:

      Creates a new, reordered particle list. One of two options can be selected. Select one particle as a start and press "To closest" to reorder the particles, starting with one particle and then continuing further to its closest neighbor. The other option is to ......?

  • Geometric Model:

    The Geometric Model tab includes all options surrounding the possible geometric models. For example, you can change the way the models are fitted to the particle lists they originate from. Additionally, the models can be populated with particles to create new particle lists based on the models.

    • Colour Options:

      Used to change the color and transparency of the geomodel.

    • Generate Points:

      Randomly generating points on a surface, sphere or line. The options and methods are further explained here artiax gen on surface and there artiax gen in surface. For some types of geomodels more options for particle population are located further down below in the geomodel tab.

    The following options depend on what kind of geomodel is currently selected.

    • Sphere Options

      Further options only applicable to line geomodels. Press "Update Sphere" to redraw the sphere after e.g. moving some of its defining particles. Press "Reorient Particles" to reorient the selected particles so that their z-axis points away from sphere.

    • Curved Line Options

      Further options only applicable to line geomodels.

      • Line Display:

        Includes various options to modify the line. Change the line thickness by varying the line radius. Press "Update Line" to redraw the line after e.g. moving some ifts defining particles or "Remove Deleted Particles" after deleting some particles of the corresponding particle list. This can also be set to happen automatically.

      • Change line fitting:

        Change the polynomial degree and resolution of the fitted line or smoothen it.

      • Move camera along line:

        Moves the camera along the line. The line is used to define the camera path, with different settings to define e.g. viewing and moving direction. Useful for recording videos in combination with the 'movie record' command.

      • Populate line with evenly spaced particles:

        Detailed option to populate the line with particles and generate a new particle list. The interparticle distance and the orientation of the particles can be defined.

    • Surface Options

      Further options only applicable to surface geomodels. Also you can update the sphere after e.g. moving some of its defining particles and reorient the selected particles so that their z-axis points along the normal of the closest part of the surface.

      • Change surface fitting:

        Change the fitting method, resolution and base level of the surface.

      • Populate surface with particles in every vertex:

        Detailed option to populate the surface with particles and generate a new particle list. Particles will be placed in every vertex. To modify the density of population, just change the resolution of the surface to lessen the amount of vertices.

    • Boundary Options

      Further options only applicable to boundary geomodels. Also you can update the boundary after e.g. moving some of its defining particles and reorient the selected particles so that their z-axis points along the average normal of the corner.

      • Change boundary fitting:

        Vary the alpha value

    • Triangulation Surface Options

      Further options only applicable to triangulation surface geomodels. Also you can update the surface after e.g. moving some of its defining particles and reorient the selected particles so that their z-axis points away from the surface.
